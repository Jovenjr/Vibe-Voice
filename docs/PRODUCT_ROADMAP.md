# Product roadmap

## P0 - Public beta usable (actual)

- Local-first sin friccion (instalacion + operacion clara).
- Remoto como modo avanzado, con hardening obligatorio.
- Scripts unificados (`start/stop/status/doctor`).
- Docs por escenario (local, remoto, control Telegram, seguridad).
- Matriz minima de smoke tests.

## P1 - Scale/public access

- Autenticacion remota nativa en la UI (token/sesion).
- Onboarding guiado de primer arranque (wizard).
- Mejor empaquetado por plataforma.
- Checklists de observabilidad y soporte operativo.
